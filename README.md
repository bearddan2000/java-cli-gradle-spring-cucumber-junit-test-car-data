# java-cli-gradle-spring-cucumber-junit-test-car-data

## Description
A POC for spring app using JUnit.
Uses cucumber plugin that publishes
report online. JUnit dumps
reports to `reports`.

## Tech stack
- java
- gradle
  - spring
  - junit
  - cucumber

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
